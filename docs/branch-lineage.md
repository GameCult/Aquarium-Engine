# Branch Lineage

Epiphany Aquarium now has four lines of work:

- `EpiphanyAquarium-Web`: React, Tauri, WebGL2, DOM interaction grammar.
- `EpiphanyAquarium-Bevy`: frozen Rust/Bevy prototype and renderer notes.
- `AquariumSynth`: procedural audio crate extracted from the Rust prototype.
- `EpiphanyAquarium-Engine`: C# engine seed for the next line.

This repo is intentionally new history. It should not pretend the C# engine
already existed inside the older commits. The older histories remain in their
own repos where their evolution makes sense.
